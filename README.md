💘 Gerador de Cantadas Nerds em C

Um pequeno programa escrito em C que gera cantadas nerds aleatórias no terminal.

A cada vez que o usuário pressiona ENTER, o programa seleciona uma cantada aleatória e exibe na tela.

Projeto criado para praticar:

Arrays em C

Funções

Geração de números aleatórios

Estruturas de repetição

Interação simples com o usuário no terminal

📂 Estrutura do Projeto
cantadas.c
cantadas (executável)

⚙️ Bibliotecas utilizadas
Biblioteca	Função
<stdio.h>	entrada / saída
<stdlib.h>	memória / aleatório
<time.h>	tempo / semente

🧠 Como o programa funciona

Um array de cantadas é criado.

O programa gera um número aleatório.

Esse número escolhe uma posição do array.

A cantada correspondente é exibida.

O usuário pressiona ENTER para gerar outra.

▶️ Compilar o programa

No terminal:

gcc cantadas.c -o cantadas

🚀 Executar
./cantadas
